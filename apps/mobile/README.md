# @mixmatch/mobile

React Native (Expo) foundation for TheMixMatch Onchain.

**This app intentionally contains no screens, navigation, or authentication
yet.** It establishes project structure, tooling, and configuration only.

## Structure

```
src/
  components/
  screens/
  navigation/
  hooks/
  services/
  utils/
  assets/
```

These folders are placeholders for future feature work.

## Local setup

```bash
cp .env.example .env
pnpm dev
```

## Testing

```bash
pnpm test
```

Runs a setup-verification test confirming the Jest/Expo testing
infrastructure works.
